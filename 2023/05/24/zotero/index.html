zotero文献管理

发表于 2023-05-24 更新于 2023-05-25 分类于 Tools

工欲善其事，必先利其器。非常好用且强大的文献管理软件，一劳永逸的设置。

插件推荐

Zotero connector 可以从直接从浏览器添加文献

Zotero translators 翻译神器，支持Google、有道等多种翻译

Zotero Tag 增加标签

Zotero Style 阅读进度条

Zotfile 高效管理PDF，重命名、移动目录等

Jasminum 中文文献管理插件

同步

在Zotfile的基础上使用坚果云同步数据、PDF文件

RSS订阅

实用功能

  • 添加文件链接作为附件
    zotero支持markdown笔记，但是不如typora使用更方便，现在可以将这两个内容结合在一起
  • 导出有标记的pdf：导出条目-包括注释
