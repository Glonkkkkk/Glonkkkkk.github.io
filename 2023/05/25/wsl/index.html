wsl的安装与使用

发表于 2023-05-25 分类于 Daily

WSL的安装可以代替虚拟机进行Linux系统的开发任务。因为被Hypatia的环境折磨很久，所以开始尝试。

准备工作

首先需要打开Windows功能：适用于Linux的Windows子系统和虚拟机平台。然后重启电脑。

版本选择

Ubuntu20.04+WSL2
这两个都可以通过MS Store来下载，注意下载的时候要关掉VPN，否则会打不开。

设置用户名和密码之后就可以使用了。打开方式有很多方式：ubuntu、WSL终端、ssh连接、命令行输入wsl

配置工作

换源：墙里的下载好慢好慢，添加清华源/中科大源可以加速下载。
然后更新sudo apt-get update && sudo apt-get upgrade

SSH连接

修改config设置 sudo vim /etc/ssh/sshd_config

重新安装ssh

sudo apt remove openssh-server # 先卸载原有的 ssh sudo apt install openssh-server # 重新安装新的 ssh

重启ssh sudo service ssh --full-restart
